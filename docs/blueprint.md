# **App Name**: LectorSync

## Core Features:

- Content Upload: Upload PDF or text content to be read.
- Customizable Reading Interface: Customize the reading layout with options to adjust font size, color themes, and page splitting.
- Google Authentication: Secure login using Google Auth provider.
- Reading Progress Sync: Save and synchronize reading progress, customizations, and bookmarks using Firestore.
- Offline Access: Enable offline reading of downloaded content.
- Page Animation: Page transition animation with several animation options to chose from

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to evoke a sense of focus and calm.
- Background color: Very light Indigo (#E8EAF6) to provide a comfortable reading experience.
- Accent color: Deep Violet (#673AB7) to highlight key interactive elements and navigation.
- Body and headline font: 'Literata', a serif font, to promote a literary and vintage aesthetic which is ideal for long-form reading. 
- Simple, clear icons for navigation and settings, ensuring ease of use.
- Clean, distraction-free layout to enhance the reading experience. Clear visual hierarchy.
- Subtle page transition animations to enhance readability and engagement without being intrusive.
